Fills the screen with (Pattern)

Parameters
[in]DeviceHandlePointer to device handle
[in]Pattern8 Bit pattern to fill the screen with.
Remarks
Pattern corresponds to 8 pixels vertically with bit 0 being the top pixel and bit 7 being the bottom.
For example, a pattern of 1 would be a line going horizontally from one side of the screen to the other every 8 pixels vertically.
